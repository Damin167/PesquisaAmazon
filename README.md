# Projeto BDD – Busca de Produtos na Amazon

## Descrição
Este projeto implementa cenários BDD utilizando Gherkin e Cucumber
para testar a funcionalidade de busca de produtos no site da Amazon Brasil.

## Tecnologias utilizadas
- Node.js
- Cucumber.js
- Selenium WebDriver
- JavaScript

## Instalação
npm install

## Execução dos testes
npm test

## Observação
Alguns cenários podem falhar devido a instabilidades, mudanças de layout
ou mecanismos de proteção do site da Amazon, o que não compromete
a estrutura BDD e a implementação proposta.
